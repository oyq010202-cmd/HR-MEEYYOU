## 💾 数据安全

### 数据存储
- **位置**: `performance_data.db`（SQLite文件）
- **大小**: 约44KB（已含示例数据）
- **格式**: 标准SQLite 3格式
- **备份**: 直接复制db文件即可

### 数据隐私
- ✅ 本地存储，不上传云端
- ✅ 不连接外部服务器
- ✅ 仅监听localhost
- ✅ 完全离线运行

### 备份建议
```bash
# 每月备份一次
cp performance_data.db backup_$(date +%Y%m).db

# 或使用系统自带的文件历史记录
```

## 🔧 系统维护

### 日常维护
1. **每月导入新数据** - 从北森系统导出并上传
2. **定期备份数据库** - 复制db文件到安全位置
3. **清理临时文件** - 删除/tmp/目录下的临时文件

### 数据清理
```bash
# 如需重置数据库（慎用！）
rm performance_data.db
python database.py
```

### 升级更新
- 保留 `performance_data.db` 文件
- 替换其他所有.py文件
- 重新启动系统
