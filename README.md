# 绩效考核管理系统 v1.0

## 📖 系统简介

这是一个基于Streamlit开发的本地绩效考核管理系统，专门针对北森系统导出的绩效数据进行分析和可视化。系统支持月度和季度考核数据，提供全方位的绩效追踪、部门分析、面谈监控等功能。

## ✨ 核心功能

### 1. 📤 数据上传中心
- 拖拽上传北森导出的Excel文件
- 自动识别考核周期（月度/季度）
- 智能数据清洗和验证
- 自动计算部门排名
- 上传历史记录查询

### 2. 👤 员工绩效追踪
- 多周期趋势分析（折线图）
- 部门排名变化追踪
- 雷达图展示各模块得分
- 单项指标明细分析
- 历史评语查看

### 3. 📈 部门绩效分析
- 自动生成部门分析报告
- 分数分布直方图
- 优秀员工识别（≥90分）
- 需关注员工提醒（<80分）
- 部门排名表

### 4. 💬 绩效面谈质量监控
- 各上级面谈完成率统计
- 未面谈员工清单
- 连续未面谈预警
- 面谈质量分析

### 5. 🔄 考核方案管理
- 记录考核方案变更
- 方案变更历史查询
- 指标权重对比

### 6. 🔍 智能查询中心
- 多维度筛选（姓名、周期、分数）
- 导出查询结果（CSV）
- 批量数据分析

## 🚀 快速开始

### 系统要求
- Python 3.8 或更高版本
- 操作系统: Windows / macOS / Linux

### 安装步骤

1. **安装依赖包**
```bash
pip install -r requirements.txt
```

2. **初始化数据库**
```bash
python database.py
```

3. **启动系统**
```bash
# Linux/macOS
chmod +x run.sh
./run.sh

# Windows
streamlit run app.py
```

4. **访问系统**
打开浏览器访问: `http://localhost:8501`

## 📊 使用流程

### 第一步: 上传数据

1. 从北森系统导出两个Excel文件:
   - `绩效结果导出.xlsx`
   - `指标评价信息导出.xlsx`

2. 在系统中进入"📤 数据上传"页面

3. 上传两个文件，点击"开始处理数据"

4. 系统会自动:
   - 识别考核周期
   - 清洗数据
   - 计算部门排名
   - 生成统计摘要

### 第二步: 查看员工绩效

1. 进入"👤 员工绩效追踪"

2. 选择员工查看:
   - 历史分数趋势
   - 排名变化
   - 各项指标得分
   - 上级评语

### 第三步: 生成部门报告

1. 进入"📈 部门绩效分析"

2. 选择考核周期和部门

3. 查看:
   - 整体统计数据
   - 分数分布
   - 优秀/需关注员工

### 第四步: 监控面谈质量

1. 进入"💬 面谈质量监控"

2. 查看:
   - 各上级完成率
   - 未完成面谈清单
   - 生成提醒

## 🗄️ 数据库结构

系统使用SQLite本地数据库，包含5个主要数据表:

1. **performance_results**: 绩效结果表
2. **performance_indicators**: 考核指标表
3. **interview_records**: 面谈记录表
4. **assessment_scheme**: 考核方案表
5. **data_upload_log**: 上传日志表

## 💡 使用技巧

### 批量导入历史数据
如果有多个月份的数据，可以按月份逐个上传，系统会自动识别周期并累积存储。

### 部门对比分析
在查询页面选择多个周期，可以进行跨期对比分析。

### 导出报告
在各个分析页面都支持截图保存，也可以使用浏览器打印功能生成PDF报告。

### 数据备份
定期备份 `performance_data.db` 文件到云盘或其他安全位置。

## 🐛 常见问题

### Q: 上传失败怎么办？
A: 检查Excel文件格式是否与北森导出格式一致，确保包含必要的列（员工、工号、总分等）。

### Q: 数据显示异常？
A: 尝试重新上传数据，或检查原始Excel文件是否有合并单元格等问题。

### Q: 如何清除数据？
A: 删除 `performance_data.db` 文件，重新运行 `python database.py` 初始化。

### Q: 支持多少数据量？
A: SQLite支持大规模数据，理论上可以存储数十万条记录，完全满足企业级使用。
