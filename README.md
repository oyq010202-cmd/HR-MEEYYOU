## 📖 项目简介

美柚绩效考核管理系统是一个**本地化部署的绩效数据分析工具**，专为补充Beisen（北森）系统在历史数据查询、多维度分析、绩效分布监控等方面的不足而设计。

### 🎯 核心价值

- **历史数据追踪**：轻松查看员工跨周期的绩效趋势，支持月度/季度/年度分别查看
- **实时分布监控**：提前预警绩效分数分布健康度，避免年终调整压力
- **智能数据分析**：多维度对比分析，为管理决策提供数据支持
- **面谈质量管理**：全面监控面谈完成率与质量，提升管理规范性
- **本地安全可控**：数据存储在本地，确保敏感信息安全

## ✨ 功能特性

### 📊 核心功能模块

| 模块 | 功能说明 | 适用场景 |
|------|---------|---------|
| **📤 数据上传** | 支持Beisen导出的Excel文件上传，自动识别周期类型 | 数据录入与管理 |

| **📊 高级分析** | 指标拆解、部门对比、贡献度分析 | 深度数据洞察 |

### 🎨 UI/UX亮点

- **美柚品牌设计**：遵循美柚品牌色彩体系（#DD5F87），企业级数据平台风格
- **响应式布局**：自适应不同屏幕尺寸，支持桌面与移动端访问
- **智能筛选器**：周期类型独立筛选，避免月度/季度/年度数据混淆
- **可视化图表**：基于Plotly的交互式图表，支持缩放、导出、悬停详情

### 🔥 独特功能

#### 1️⃣ 周期类型分离分析
```
传统方式：月度、季度、年度数据混在一起，趋势混乱
本系统 ：独立筛选，清晰追踪同类周期表现
```

#### 2️⃣ 多周期平均分布预判
```
基于员工最近N期平均分，提前预判年度绩效格局
支持强制分布适配分析，避免年终被动调整
```

#### 3️⃣ 面谈质量深度监控
```
- 上级维度趋势分析（完成率时间序列）
- 面谈质量分析（面谈时长、反馈质量）
- 风险预警（低完成率上级识别）
- 上级排名（绩效管理能力评估）
```

## 🚀 快速开始

### 📋 环境要求

- **操作系统**：Windows 10/11, macOS, Linux
- **Python版本**：3.8 或更高
- **浏览器**：Chrome, Edge, Safari（推荐Chrome）

### 📦 安装步骤

#### 方法1：一键安装（推荐）

1. **下载项目文件**
   ```
   绩效系统/
   ├── app.py
   ├── database.py
   ├── data_processor.py
   ├── visualizations.py
   ├── meiyou_logo.png
   ├── .streamlit/
   │   └── config.toml
   ├── requirements.txt
   └── README.md
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **启动系统**
   ```bash
   streamlit run app.py
   ```

4. **访问系统**
   - 系统会自动打开浏览器
   - 默认地址：`http://localhost:8501`

#### 方法2：批处理启动（Windows）

创建 `run.bat` 文件：
```batch
@echo off
echo 正在启动绩效考核管理系统...
streamlit run app.py
pause
```

双击 `run.bat` 即可启动。

## 📚 使用指南

### 1️⃣ 数据上传

#### 准备数据文件
从Beisen系统导出两个文件：
- **绩效结果文件**：包含总分、等级、系数等
- **指标评价文件**：包含各考核指标的详细分数

#### 上传流程
1. 进入"📤 数据上传"页面
2. 选择两个Excel文件
3. （可选）手动指定考核周期，如"2026年3月月度考核"
4. 点击"🚀 开始处理"
5. 系统自动识别周期类型（月度/季度/半年度/年度）

#### 数据验证
- ✅ 自动去重（同一员工+同一周期只保留最新记录）
- ✅ 数据完整性校验
- ✅ 周期类型自动识别

---

### 2️⃣ 员工绩效追踪

**使用场景**：1on1面谈、绩效回顾、转正评审

**操作步骤**：
1. 选择**周期类型**（月度/季度/年度）
2. 选择**部门** → **员工**
3. 查看绩效趋势图、指标雷达图

**关键指标**：
- 📈 绩效趋势曲线
- 🎯 各指标得分对比
- 📊 排名变化追踪

**实用技巧**：
- 季度数据：适合季度回顾、晋升评估
- 年度数据：适合年度总结、长期发展规划

---

### 3️⃣ 绩效分布监控

#### Tab1：当前周期分布
监控单个周期的分数分布健康度

**健康度指标**：
- 🟢 优秀（90+分占比 > 15%）
- 🟡 良好（标准差 > 8）
- 🔴 需关注（低分人数过多）

#### Tab2：多周期平均分布
基于最近N期平均分，提前预判年度绩效

**应用场景**：
- 年中预判年终绩效格局
- 提前发现"躺平风险"员工
- 识别持续高绩效者

#### Tab3：部门分布对比
对比不同部门的评分标准

**关键发现**：
- 评分偏松的部门（平均分高、标准差小）
- 评分偏严的部门（平均分低）
- 区分度最好的部门（标准差大）

#### Tab4：强制分布适配
预判当前分数能否满足强制分布要求

**配置项**：
- 目标比例：A级20% / B级50% / B-级20% / C级10%
- 分析周期：最近N期平均分
- 筛选维度：部门、年份

---

### 4️⃣ 面谈质量监控

**4个维度全面监控**：

| Tab | 监控内容 | 关键指标 |
|-----|---------|---------|
| 📈 上级趋势分析 | 各上级的面谈完成率时间序列 | 完成率趋势 |
| 🎯 面谈质量分析 | 面谈时长、反馈质量评估 | 质量得分 |
| ⚠️ 风险预警 | 低完成率上级识别 | 连续未完成 |
| 🏆 上级排名 | 绩效管理能力综合评估 | 综合排名 |

---

### 5️⃣ 智能查询

**多维度组合查询**：
- 周期类型：月度/季度/年度
- 员工信息：姓名/工号模糊搜索
- 考核周期：多选周期
- 分数范围：滑块筛选

**查询结果**：
- 表格展示：支持排序、导出
- 统计摘要：自动汇总关键指标

---

### 6️⃣ 高级分析

#### 指标拆解分析
发现评分偏高/偏低的指标，识别区分度不足的问题

#### 部门/岗位对比
多维度横向对比，找出管理短板

#### 指标贡献分析
计算各指标对总分的贡献度，优化考核权重

## ❓ 常见问题

### Q1: 系统启动后浏览器没有自动打开？

**A**: 手动访问 `http://localhost:8501`

### Q2: 上传文件后提示"格式错误"？

**A**: 检查以下几点：
- 确保是从Beisen导出的原始Excel文件
- 文件未被手动编辑过
- 必需列完整存在

### Q3: 数据上传成功，但查询不到？

**A**: 
1. 检查周期类型筛选器是否选对
2. 确认数据库中period_type字段值正确
3. 尝试选择"全部"周期类型查看

### Q4: 如何删除错误上传的数据？

**A**: 
1. 进入"📋 数据管理"页面
2. 找到对应周期的数据
3. 点击"删除"按钮

### Q5: 系统可以多人同时使用吗？

**A**: 
- **本地模式**：仅支持单用户
- **网络模式**：可部署到服务器，支持多人访问（需额外配置）

### Q6: 数据存储在哪里？是否安全？

**A**: 
- 数据存储在本地SQLite数据库（`performance_data.db`）
- 不会上传到云端
- 建议定期备份数据库文件

### Q7: 如何备份数据？

**A**: 
1. 直接复制 `performance_data.db` 文件
2. 或在"数据管理"页面导出为Excel

### Q8: 报错"name 'tab1' is not defined"？

**A**: 
- 下载最新版本的 `app.py`
- 清除Python缓存：`del /s /q __pycache__`
- 重启系统

### Q9: 周期类型筛选后显示"暂无数据"？

**A**: 
- 检查数据库中是否有该周期类型的数据
- 确认 `database.py` 已更新到最新版本（包含中英文映射）

### Q10: 如何修改品牌颜色？

**A**: 
编辑 `.streamlit/config.toml` 文件中的 `primaryColor` 值
